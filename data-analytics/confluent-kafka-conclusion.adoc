= 結論
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
本文檔提供了將 Confluent 分層儲存與NetApp儲存體結合使用的最佳實務指南，包括驗證測試、分層儲存效能結果、調整、Confluent S3 連接器和自平衡功能。考慮到 ILM 策略、經過多項效能測試驗證的 Confluent 效能以及業界標準的 S3 API， NetApp StorageGRID物件儲存是 Confluent 分層儲存的最佳選擇。
